# Just an exercise in using 'let', where my favorite food changes over time

```
let favFood = 'Beets';
console.log(favFood); // should return beets
favFood = 'pizza'
console.log(favFood); // should return pizza
favFood = 'tuna'
console.log(favFood); // should return tuna
```
